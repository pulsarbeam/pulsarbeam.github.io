Arrays vs Objects
Objects are used to represent things in your code anything that can be defined by a set of characteristics, these are called properties and consist of a key and a value. To access these properties we use either dot or bracket notation

We use arrays when we want to store a list of multiples items in a single variable they can be useful when creating ordered list as items in the list can be accessed by their numerical values. Arrays use zero-based indexing so the first item in an array has an index of 0, the next item will have and index of 1 and so on

Objects represent things with properties while arrays create and store lists of data in a single variable.

What Are Functions?
Functions are a block of reusable code written to perform tasks. A function consists of a set of statements but executes as a single unit. to use a function you need to use the function keyword followed by the name of the function, parentheses, and the body of the function wrapped in curly brackets.
